# Сайт-визитка клининговой компании

Проект представляет собой многостраничный сайт-визитку для клининговой компании ООО «КлинМастер».

## Основные страницы

- `index.html` — главная страница сайта
- `services.html` — страница со списком услуг
- `flat.html` — страница услуги по уборке квартиры
- `house.html` — страница услуги по уборке дома
- `about.html` — страница с информацией о компании
- `privacy.html` — страница политики конфиденциальности
- `404.html` — страница ошибки 404

## Используемые технологии

- HTML5
- CSS3
- JavaScript
- PHP
- Яндекс.Карты

## Структура проекта

```text
assets/css/style.css      Основные стили сайта
assets/js/script.js       JavaScript-логика сайта
assets/images/            Изображения проекта
handler.php               Обработчик формы заявки
```

## Основная функциональность

- адаптивная верстка;
- модальная форма заявки;
- маска ввода телефона;
- мобильное бургер-меню;
- кнопка возврата наверх;
- переключение светлой и темной темы;
- плавающая кнопка связи через мессенджеры;
- встроенная Яндекс.Карта.

## Запуск проекта

Для просмотра клиентской части достаточно открыть файл `index.html` в браузере.

Для проверки PHP-обработчика формы необходимо запустить проект через локальный сервер или разместить его на хостинге с поддержкой PHP.
